# DPD for Apple Dictionary

Dear friends in the Dhamma,

DPD is now available as a native Apple Dictionary on macOS. This means you can look up Pāḷi words directly in the built-in Dictionary app — no extra software to install, no browser tabs to switch between.

The most useful feature is system-wide lookup. Select any Pāḷi word in Safari, Preview, Pages, or any macOS application, press Control + Command + D, and a popup definition appears instantly. DPD recognizes inflected forms, so looking up "dhammassa" or "dhammesu" will take you straight to the main "dhamma" entry.

To install, download dpd-apple.dictionary.zip from the GitHub releases page (https://github.com/digitalpalidictionary/dpd-db/releases/latest), unzip it, and move the dictionary file to ~/Library/Dictionaries/. Full step-by-step instructions are available in the installation guide (https://digitalpalidictionary.github.io/dpd-db/install/apple_dict/).

The Apple Dictionary is updated with every new DPD release, so you will always have access to the latest entries and corrections. If you study Pāḷi on a Mac, this is a convenient addition to your toolkit.
