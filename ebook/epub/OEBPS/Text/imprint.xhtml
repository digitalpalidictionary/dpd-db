Digital Pāḷi Dictionary

DPD is a feature-rich Pāḷi-English dictionary for Win, Mac, Linux, iOS, and Android which runs in GoldenDict, MDict, or any other application that supports those formats.

This light version of DPD has been specially adapted for Kindle.

For more information please visit the website digitalpalidictionary.github.io
