# Project Guide

## What It Is And Why
This project is an ongoing system for entering dictionary data through a GUI, testing that data, generating the database, and exporting it into multiple formats. Its purpose is to give contributors a reliable workflow for maintaining and releasing the Digital Pāḷi Dictionary.

## Who It Is For
This project is for internal contributors and editors who maintain the dictionary data and release pipeline.

## One-Off Or Ongoing
This is an ongoing project. It supports continuous dictionary maintenance, validation, database generation, and export work over time.

## What It Will Produce
- updated dictionary data entered through the GUI
- tested and validated database content
- generated database outputs
- export artifacts in multiple formats

## How You'll Know It Worked
The project is successful when contributors can reliably enter data in the GUI, run tests, generate the database, and export release artifacts without manual workaround steps.
