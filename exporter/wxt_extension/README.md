# DPD WXT Extension

This directory contains the source code for the Digital Pāḷi Dictionary browser extension, rebuilt using the WXT framework for cross-browser compatibility (Chrome and Firefox).

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

## Build

To build for production:

```bash
npm run build
```
